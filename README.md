## 데이터가 암호화된 블로그
### 1) 암호화?
> 데이터가 암호화되어 웹 데이터베이스에 저장됨.
> 권한이 있는 로그인된 사용자가 아니면 수정과 저장할 수 없음
> 암호화된 데이터는 F12등의 개발자 도구를 통해서 확인할 수 없음
> 암호화된 데이터는 크롤링을 통해 확인할 수 없음
> 암호화된 데이터는 Github에 저장되지 않음

### 2) 코드설명
> 0 : 방문자용과 유저용이 똑같음  
> 1 : 유저만 열람 가능  
> 2 : 방문자용과 유저용이 다름  
> 3 : 방문자와 유저 모두 열람 불가
